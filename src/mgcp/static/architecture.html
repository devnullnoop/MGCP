Technology Stack

Qdrant Vector search, local or server mode. BGE embeddings (768 dim).
NetworkX In-memory graph. Typed relationships, Louvain community detection.
SQLite Async via aiosqlite. Lessons, contexts, telemetry, communities.
FastMCP MCP server framework. 38 tools exposed to LLM clients.
sentence-transformers BAAI/bge-base-en-v1.5 model. Instruction-prefixed queries.
FastAPI Web dashboard with WebSocket updates and graph visualization.
